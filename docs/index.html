Three Product Views, One Simulator

Switch between Claude Chat, Claude Code, and Claude Cowork to build demos for any product surface.

Claude Chat

Simulate the full claude.ai experience with conversation threads, markdown rendering, and artifact panels. Includes realistic typing indicators and streaming responses.

Claude Code

Render a terminal CLI experience with prompt input, streaming output, tool-use blocks, and file diffs. Ideal for developer-focused demos and presentations.

Claude Cowork

The autonomous desktop agent with task dashboards, progress indicators, and multi-step workflows. Shows Claude working independently on complex tasks.

Built for Realistic Demos

Everything you need to build convincing demos of Claude in action.

Realistic Streaming Playback

Text streams character-by-character with natural pacing, just like the real Claude products. Typing effects, thinking indicators, and configurable playback speed let you fine-tune the demo experience.

  • Character-by-character streaming with natural timing
  • Thinking and processing indicators
  • Adjustable playback speed (0.5x to 4x)

Visual Simulation Editor

Design simulations visually with a timeline-based editor. Drag events from the palette, configure properties in the detail panel, and preview changes in real time.

  • Timeline-based event sequencing
  • Event palette with drag-and-drop
  • JSON import & export for sharing

Simulated Chrome Browser

HTML artifacts open in a realistic Chrome window, complete with address bar and tab styling. Show full web applications and interactive previews inside the simulation.

  • Accurate Chrome window rendering
  • Live HTML artifact rendering
  • Address bar with realistic URL display

Get Started in Minutes

Clone the repo, install dependencies, and start building simulations locally.

Prerequisites: Node.js 18+ and npm
Terminal
$ git clone https://github.com/versantus/model-reel.git
$ cd model-reel
$ npm install
$ npm run dev
VITE v8.x ready in 400ms
➔ Local: http://localhost:5173

Open http://localhost:5173 in your browser

The simulator launches with a demo selection screen. Pick any of the 10 built-in simulations or create your own from the editor.
